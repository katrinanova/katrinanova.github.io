- Work -

ColorNote

A web application for creating, organizing and sharing notes freely.

Each note can have multiple file attachments that are stored in Amazon S3.

The back-end is a RESTful JSON API in Rails with client side interface provided mainly by Backbone.js. It has custom SQL queries and overrides Backbone’s parse method to load all associations in a single AJAX request.

ColorNote allows users to sign in with Twitter or Facebook. There user-friendly interface features auto-save and the ability to organize notes by colors.

keeping up with the ASTEROIDS

A browser game inspired by the classic arcade game and current culture issues.

This game relies on radial geometry and basic trigonometry to dynamically move, rotate and accelerate objects.

The player has 3 lives to shoot all the asteroids, after initial hit asteroids break into smaller pieces.

Chess

Chess game made to be played in the terminal. Can be played with one or two players.

Computer player AI finds best move recursively by simulating opponent’s moves.

The game utilizes strict class inheritance to maintain DRY code.
