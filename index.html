ColorNote

| Live | Github

A web application for creating, organizing and sharing notes freely. Each note can have multiple file attachments that are stored in Amazon S3.

The back-end is a RESTful JSON API in Rails with client side interface provided mainly by Backbone.js.

ColorNote allows users to sign in with Twitter or Facebook. There user-friendly interface features auto-save and the ability to organize notes by colors.

A browser game inspired by the classic arcade game and current culture issues.
